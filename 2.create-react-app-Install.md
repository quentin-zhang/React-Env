# 2.create-react-app环境安装-Windows

（1）安装create-react-app

```bash
npm install -g create-react-app
```

（2）新建App

```bash
     cd d:\
     mkdir apps 
     cd d:\apps
     npx create-react-app my-app
```

（3）开启web站点

```bash
      cd my-app
      npm start
```

（4）访问3000端口，查看页面

```bash
http://localhost:3000/
```
